1. SYSTEM OVERVIEW
================================================================================

The Task Tracking System is a web-based application built with PHP and MySQL
deployed on Railway

MAIN FEATURES:
- User Registration with validation
- Secure Login/Logout with sessions
- Dashboard to view all user tasks
- Add new tasks
- Edit existing tasks
- Delete tasks with confirmation
- Toggle task status (Pending/Complete)


================================================================================
2. DATABASE STRUCTURE
================================================================================

DATABASE NAME: trackersystemdb

TABLE 1: Users
--------------
UserID          INT             Primary Key, Auto Increment
Username        VARCHAR(50)     Unique, Not Null
uPassword       VARCHAR(255)    Not Null (stores hashed password)
Email           VARCHAR(100)    Unique, Not Null
CreatedAt       TIMESTAMP       Default: Current Timestamp

TABLE 2: Tasks
--------------
TaskID          INT             Primary Key, Auto Increment
UserID          INT             Foreign Key -> Users(UserID), Not Null
TaskTitle       VARCHAR(200)    Not Null
TaskDescription TEXT            Nullable
TaskStatus      ENUM            Values: 'Pending', 'Complete', Default: 'Pending'
CreatedAt       TIMESTAMP       Default: Current Timestamp
UpdatedAt       TIMESTAMP       Auto-update on modification

RELATIONSHIPS:
- One User can have Many Tasks (1:N relationship)
- Tasks.UserID references Users.UserID
- ON DELETE CASCADE: When a user is deleted, all their tasks are deleted

3. FILE STRUCTURE
================================================================================

According to hierarchy
Index = Main start of the website(auto redirects to login page)
    Login = Main entry point for user login, directs to dashboard if account exists, goes to register if not.
        Register = user registers if there is no account yet, after register goes back to Login.
      
      
    Dashboard = Main Page for the utilities
        Add Task = adds task seen in dashboard
        Edit Task = edits task, result seen in dashboard
        Delete Task = removes task
        Toggle Task(Pending/Complete) = toggles a task and tags them either pending or complete
    
    Logout = logs out the user from the website and redirects back to login page.

Miscellaneous:
